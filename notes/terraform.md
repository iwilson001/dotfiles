# How to run

`terraform init`
initialize terraform

`terraform plan`
scaffolds infra to be built

`terraform apply`
applies plan

`terraform show`
shows output

`terraform destroy`
destroys infra

# Troubleshooting

- 400
 - Did you make sure to run with `CLOUDFLARE_API_TOKEN=... <your_command>`

- 403
 - Does the api token have proper permissions?
